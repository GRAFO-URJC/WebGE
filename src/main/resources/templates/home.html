What is WebGE?

WebGE is a tool for the construction of algorithms capable of working with Evolutive Grammars. Developed by researchers from the Rey Juan Carlos University. This initiative aims to encourage the use of this tool, providing a web format in which users can interact with each other and, in addition, make experiments with their own Evolutionary Grammars
